# AI vs ML vs DL

## 이 문서에서 배우는 것
- AI, 머신러닝, 딥러닝의 차이를 구분한다.
- 세 개념이 포함 관계라는 점을 이해한다.
- LLM이 어디에 속하는지도 연결해서 본다.
- 실무에서 왜 이 구분이 중요한지 감을 잡는다.

## 왜 중요한가
입문자가 가장 먼저 헷갈리는 부분이 바로 이 구분이다. "AI를 쓴다"는 말은 너무 넓고, 실제로는 규칙 기반 시스템인지, 전통적인 머신러닝인지, 딥러닝인지, LLM 기반인지까지 내려가야 의미가 분명해진다.

이 차이를 알아야 문서를 읽을 때도 맥락이 보인다. 예를 들어 누군가 "이 문제는 꼭 딥러닝까지 갈 필요가 없다"고 말한다면, 그 뜻은 단순한 구조화 데이터 문제에는 더 가벼운 머신러닝이 충분할 수 있다는 의미다.

## 핵심 개념 설명
AI는 가장 큰 개념이다. 사람이 하던 지능적 작업을 기계가 하도록 만드는 전체 분야를 말한다. 머신러닝은 그중에서도 데이터를 보고 패턴을 학습하는 접근이다. 딥러닝은 머신러닝 안에서 여러 층의 신경망을 이용해 복잡한 패턴을 학습하는 방식이다.

정리하면 `AI > ML > DL` 구조다. 그리고 LLM은 딥러닝 기반의 대규모 언어 모델이므로 DL 아래에 놓인다.

비슷한 개념 차이:

| 개념 | 질문 | 핵심 | 대표 예시 |
| --- | --- | --- | --- |
| AI | 기계를 어떻게 똑똑하게 만들까? | 전체 분야 | 규칙 기반 시스템, 검색, ML, DL |
| ML | 데이터를 통해 어떻게 배우게 할까? | 학습 방법 | 스팸 분류, 수요 예측 |
| DL | 복잡한 패턴을 어떻게 더 잘 배우게 할까? | 신경망 기반 방법 | 이미지 인식, 음성 인식, LLM |

핵심은 "딥러닝이 AI의 전부는 아니다"라는 점이다. 추천 시스템, 검색 랭킹, 이상 탐지처럼 전통적인 머신러닝이 여전히 강한 영역도 많다.

## 작은 사례로 보기

### 사례 1: 고객 문의 자동 분류
- AI 관점: 사람이 하던 분류 작업을 자동화한다.
- ML 관점: 과거 문의 데이터와 분류 라벨을 보고 패턴을 학습한다.
- DL 관점: 문의 텍스트를 신경망이 직접 표현 학습해 분류한다.

### 사례 2: 챗봇
- AI 관점: 사용자 질문에 자연스럽게 응답한다.
- ML 관점: 언어 데이터를 학습한 모델을 사용한다.
- DL 관점: 트랜스포머 기반 LLM이 토큰을 예측하며 답을 생성한다.

## 예시 / 비유
AI를 요리 전체라고 보면, 머신러닝은 요리법을 사람이 일일이 쓰지 않고 데이터를 보며 감을 익히는 방식이다. 딥러닝은 더 복잡한 재료 조합과 조리 패턴을 다룰 수 있는 고급 조리 시스템에 가깝다.

또 다른 비유로 보면:
- AI는 회사 전체
- ML은 데이터 팀의 업무 방식
- DL은 그 안의 대규모 신경망 팀
- LLM은 언어 처리에 특화된 제품 라인

## 한눈에 정리
- AI: 큰 우산
- ML: 데이터로 학습하는 방법
- DL: 신경망을 활용한 ML의 한 종류
- LLM: DL 기반 언어 모델

## 실무에서 왜 구분해야 하나
이 구분은 기술 선택과 비용 판단에 바로 연결된다.
- 표 형태 매출 데이터 예측이라면 전통적 ML이 더 빠르고 싸게 끝날 수 있다.
- 이미지 분류나 음성 인식은 DL이 강하다.
- 자연어 생성, 요약, 질의응답은 LLM이 유리하다.

즉, "최신 기술"을 고르는 것이 아니라 "문제에 맞는 기술 레이어"를 고르는 감각이 중요하다.

## 자주 하는 오해
- AI와 ML을 같은 말로 쓰는 경우가 많다.
- 머신러닝은 옛날 기술, 딥러닝은 최신 기술이라고 단순화하면 안 된다.
- 딥러닝이 항상 더 좋은 것은 아니다. 데이터, 비용, 문제 유형에 따라 다르다.
- LLM을 쓰면 모든 자연어 문제가 자동으로 해결된다고 보면 위험하다.

## 체크포인트
- AI, ML, DL의 포함 관계를 말할 수 있는가?
- 딥러닝이 머신러닝의 한 종류라는 점을 설명할 수 있는가?
- LLM이 왜 딥러닝 아래에 들어가는지 설명할 수 있는가?
- 같은 문제에 대해 ML과 DL 중 무엇을 택할지 직관적으로 말할 수 있는가?

## 한 줄 요약
AI는 큰 분야이고, ML은 그 안의 학습 방식이며, DL은 ML 안의 신경망 기반 접근이고, LLM은 그 딥러닝의 언어 모델 계열이다.
