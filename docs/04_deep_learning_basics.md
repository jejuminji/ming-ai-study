# 딥러닝 기초

## 이 문서에서 배우는 것
- 딥러닝이 왜 등장했고 무엇을 잘하는지 이해한다.
- 인공신경망, weight, bias, 활성화 함수, 손실 함수, 역전파의 관계를 본다.
- MLP, 분류 문제, 최적화, 초기화가 왜 중요한지 큰 그림을 잡는다.
- CNN, RNN, GAN 같은 대표 아키텍처가 어떤 문제를 해결하려고 나왔는지 이해한다.

## 왜 중요한가
이미지, 음성, 언어 같은 복잡한 데이터를 잘 다루는 현대 AI의 중심에는 딥러닝이 있다. LLM도 결국 딥러닝 위에 세워져 있기 때문에, 이 기본 원리를 알면 이후 문서가 훨씬 자연스럽게 읽힌다.

또한 딥러닝은 단순히 "층이 많은 모델"이 아니다. 신경망을 함수로 보고, weight와 bias가 어떤 의미인지 이해하고, 경사하강법과 역전파로 어떻게 학습하는지를 이해해야 실제로 모델이 왜 작동하는지 보이기 시작한다.

## 핵심 개념 설명
딥러닝은 여러 층의 신경망을 사용해 입력에서 점점 더 추상적인 표현을 학습한다. 각 뉴런은 입력을 받아 가중치(weight)와 편향(bias)을 적용하고, 활성화 함수를 거쳐 다음 층으로 전달한다. 모델은 예측과 정답의 차이를 손실 함수로 계산하고, 역전파를 통해 가중치를 조정한다.

쉬운 설명으로 말하면, 딥러닝은 "정답과의 차이를 줄이는 방향으로 내부 연결을 계속 조정하는 시스템"이다.

여기서 입문자가 꼭 잡아야 할 질문은 다음과 같다.
- weight와 bias는 무엇을 조절하는가?
- 선형 모델만으로는 왜 부족한가?
- 왜 비선형 활성화 함수가 필요한가?
- 역전파는 왜 필요한가?
- 왜 초기화와 최적화기가 학습 결과를 바꾸는가?

전통적 머신러닝과 비교:

| 항목 | 전통적 ML | 딥러닝 |
| --- | --- | --- |
| 특징 추출 | 사람이 많이 설계 | 모델이 많이 학습 |
| 데이터 요구량 | 비교적 적을 수 있음 | 보통 더 많음 |
| 계산 비용 | 비교적 낮음 | 높음 |
| 강점 | 구조화 데이터 | 이미지, 음성, 텍스트 |

대표 구조도 이름만 외우기보다 역할로 이해하는 편이 좋다.

| 구조 | 왜 나왔나 | 대표 활용 |
| --- | --- | --- |
| MLP | 기본적인 다층 신경망 | 표 형태 데이터, 기초 분류 |
| CNN | 공간 구조를 잘 다루기 위해 | 이미지, 영상 |
| RNN | 순서 정보를 다루기 위해 | 시계열, 초기 NLP |
| GAN | 생성과 판별을 경쟁시키기 위해 | 이미지 생성 |

딥러닝의 중요한 이론 포인트도 있다. Universal Approximation Theorem은 충분한 신경망이 매우 다양한 함수를 근사할 수 있다는 뜻을 말하지만, 이것이 "아무 신경망이나 쉽게 잘 학습된다"는 뜻은 아니다. 실제 학습에서는 기울기 소실, 초기화 문제, 과적합, 계산 비용 같은 현실적인 한계가 계속 등장한다.

## 예시 / 비유
사진 속 고양이를 찾는 문제에서 전통적 ML은 귀 모양, 눈 위치 같은 특징을 사람이 정할 수 있다. 딥러닝은 이런 특징 표현을 모델이 계층적으로 스스로 배운다. MLP는 기본 뼈대이고, CNN은 이미지처럼 위치 관계가 중요한 데이터를 더 잘 다루도록 나온 특화 구조라고 볼 수 있다.

## 한눈에 정리
- 인공신경망은 입력을 함수처럼 변환하는 구조다.
- weight와 bias는 "어떻게 변환할지"를 학습하는 핵심 파라미터다.
- 활성화 함수는 선형 모델 한계를 넘게 해 준다.
- 역전파와 경사하강법은 학습의 핵심 루프다.
- CNN, RNN, GAN은 서로 다른 데이터 문제를 해결하기 위해 발전한 구조다.

## 자주 하는 오해
- 층이 많으면 무조건 좋은 모델이 되는 것은 아니다.
- 딥러닝은 마법이 아니라 많은 데이터와 계산으로 패턴을 학습하는 방법이다.
- Universal Approximation Theorem이 있다고 해서 학습이 항상 쉬운 것은 아니다.
- CNN, RNN, GAN은 서로 경쟁하는 최신 유행어가 아니라, 서로 다른 문제를 해결하려고 나온 구조다.

## 체크포인트
- weight와 bias를 직관적으로 설명할 수 있는가?
- 왜 비선형 활성화 함수가 필요한지 말할 수 있는가?
- 손실 함수, 경사하강법, 역전파의 관계를 설명할 수 있는가?
- CNN, RNN, GAN이 각각 어떤 문제에 잘 맞는지 구분할 수 있는가?

## 한 줄 요약
딥러닝은 신경망을 함수처럼 학습시키는 방법이며, weight·bias·활성화·역전파·아키텍처 역할을 함께 이해해야 큰 그림이 잡힌다.
