# OpenAI Python 예제 모음

이 폴더는 공식 OpenAI 문서를 참고해, 쉬운 순서대로 따라칠 수 있는 Python 예제를 정리한 공간입니다.

## 파일 읽는 순서

1. `기초 텍스트 응답-01.py`
2. `구조화된 출력-02.py`
3. `이미지 이해-03.py`
4. `함수 호출-04.py`

## 실행 전 준비

- Python 3.10+
- `pip install openai`
- 환경 변수 `OPENAI_API_KEY` 설정

## 참고한 공식 문서

- Responses API: https://platform.openai.com/docs/api-reference/responses/create?lang=python
- Structured Outputs: https://platform.openai.com/docs/guides/structured-outputs?api-mode=responses&lang=python
- Vision: https://platform.openai.com/docs/guides/vision?lang=python
- Function Calling: https://platform.openai.com/docs/guides/function-calling?api-mode=responses&lang=python

## 예제 구성 원칙

- 처음 보는 사람도 읽을 수 있도록 쉬운 순서로 배치
- 코드 줄마다 역할이 보이도록 자세한 주석 추가
- 실무 감각을 위해 "왜 이렇게 쓰는지"도 함께 설명
